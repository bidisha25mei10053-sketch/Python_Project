## Overview
A fun and interactive Python game where the computer picks a random number between 1 and 100, and the player tries to guess it with hints provided after each attempt.

## Problem Statement
People enjoy quick logic games that challenge their intuition. This project provides entertainment while demonstrating Python basics like loops, conditionals, and random number generation.

## Features
- Random number generation
- Interactive hints ("Too high" / "Too low")
- Counts number of attempts
- Option to play multiple rounds

## Tools & Concepts Used
- Python 3
- Built-in random module
- Loops, conditionals, and user input
